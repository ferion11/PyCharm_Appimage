# PyCharm_Appimage

> PyCharm Appimage for linux 64bits

####  1- Download "PyCharm-VERSION-x86_64.AppImage"
####  2- Make executable:
```
$ chmod +x PyCharm-VERSION-x86_64.AppImage
```
####  3- Run it:
```
$ ./PyCharm-VERSION-x86_64.AppImage
```
